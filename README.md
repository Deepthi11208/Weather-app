Set-Content README.md "# Weather App

A real-time weather app built with HTML, CSS, and JavaScript using OpenWeatherMap API.

## Features
- Search any city worldwide
- Shows temperature, humidity, wind speed and pressure
- Beautiful dark theme UI

## How to run
1. Clone the repo
2. Open index.html in browser

## Tech Stack
- HTML
- CSS
- JavaScript
- OpenWeatherMap API"
